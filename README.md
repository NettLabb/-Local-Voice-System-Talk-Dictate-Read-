# -Local-Voice-System-Talk-Dictate-Read-
🚀 What it does  This project is a local voice system that runs entirely on your computer.  It supports:  🎙️ Talk → voice conversation style interaction 📝 Dictate → speech-to-text input (voice → text) 📖 Read → text-to-speech output (text → voice)  No internet required for core features.
